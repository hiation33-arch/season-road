## 프로젝트 소개

Season Road는 계절별 여행 트렌드와 관광 데이터를 결합해 최적의 여행 코스를 추천하는 싱글 페이지 웹 서비스입니다.  
드라마 촬영지, 무장애 코스, 한류 트렌드 키워드까지 아우르며 누구나 손쉽게 여행을 계획할 수 있도록 돕습니다.

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| 🗺️ 카카오맵 여행 코스 | 당일 / 1박 / 2박 코스를 핀으로 구분 표시 |
| 🌸 계절 필터 | 봄·여름·가을·겨울 계절별 코스 추천 |
| 📈 트렌드 연동 | 네이버 데이터랩 인기 키워드 기반 여행지 반영 |
| 🎬 한류 촬영지 | K-드라마·영화 촬영지 검색 및 코스 제안 |
| ♿ 무장애 코스 | 이동 약자를 위한 배리어프리 여행 코스 제공 |

---

## 기술 스택

- **Frontend** : HTML5 / CSS3 / JavaScript (단일 파일 구성)
- **지도** : 카카오맵 API
- **데이터** : 한국관광공사 TourAPI, 네이버 데이터랩 API
